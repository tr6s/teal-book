## この章でできるようになること

- 自動タスクが何を自動化してくれるかがわかる
- 役割と紐づけて新しい自動タスクを作成できる
- 実行結果の通知先（Web ダッシュボード / LINE / Slack）を選べる

![自動タスクのタイムライン](/images/teal-user-guide/08-auto-tasks-cron.png)

## 自動タスクとは

**自動タスク** は、一度設定するだけで AI が定期的にタスクを実行する機能です。「毎朝8時に競合他社の X 投稿を分析してレポート」「毎週月曜にメールアドレスを抽出して保存」など、決まった時刻・頻度で AI が自律的に動きます。

通常の会話では「ユーザーが指示 → AI が応答」という1往復ですが、自動タスクでは **指示は最初の1回だけ** で、以降は AI が時間軸に沿って動き続けます。Teal が「実行できる AI」と呼ばれる理由のひとつです。

## 自動タスクの例

| タスク名 | 実行内容 | 頻度の例 |
| --- | --- | --- |
| X 自動投稿 | 指定テーマで X（旧Twitter）に自動投稿 | 毎日 / 週数回 |
| 競合分析レポート | 競合他社の X 投稿や Web を分析 | 毎日 / 週次 |
| 朝のブリーフィング | 天気・ニュース・予定をまとめて通知 | 毎朝 |
| メールアドレス保存 | Gmail からアドレスを抽出して保存 | 自動 |
| 業界別ターゲット収集 | 特定地域・業種の法人リストを収集 | 定期実行 |

これらは出発点の例です。自分の業務に合わせて、自由に作れます。

## 操作手順

### Step 1. 役割を準備する

自動タスクは「役割」と紐づきます。先に第06章で扱った役割を1つ作っておくと、自動タスクの設定がスムーズです。

たとえば「X 自動投稿」を作りたい場合は、「X 自動投稿」という役割を先に作り、投稿のトーン・文字数・ハッシュタグの方針などを役割の指示文に書いておきます。

### Step 2. 自動タスクを新規作成する

1. Web ダッシュボードのサイドバーから「役割」を開きます
2. 紐づけたい役割の詳細画面で「自動タスク」タブを開きます
3. 「+ 新しい自動タスク」をタップして、以下を入力します
   - タスク名（例: 平日朝のX投稿）
   - 実行の指示文（例: 業界トピックを1つ選び、120字程度で投稿文を作成）
   - スケジュール（毎日 / 平日 / 特定の曜日 など）
   - 実行時刻（例: 朝 7:30）

### Step 3. 通知先を選ぶ

自動タスクの実行結果は、Web ダッシュボードに残ると同時に、選んだチャネルに通知が届きます。

| 通知先 | 向いている場面 |
| --- | --- |
| Web ダッシュボード | 後でまとめて確認したい / ログとして残したい |
| LINE | スマホで素早く確認したい / 一人で運用 |
| Slack | チームで結果を共有したい |

複数の通知先を同時に選ぶこともできます。たとえば「自分の確認用に LINE、チーム共有用に Slack」のような併用です。

### Step 4. 実行と確認

設定が終わると、指定した時刻に AI が自動でタスクを実行します。

1. 実行が終わると、Web ダッシュボードの会話履歴に新しいエントリが追加されます
2. 通知先に指定した連携先（LINE / Slack など）にも結果が届きます
3. 必要に応じて、実行結果の会話を開いて続きを指示できます（「もう少し短く」「別案を3つ」など）

「自動で動く」と「人間が介入できる」の両方が両立する設計です。完全な無人化ではなく、AI が下書きを作り、人間が必要なときだけ介入する、という運用が現実的です。

## つまずきやすいポイント

:::message
自動タスクは **役割と紐づいている** ため、役割を削除するとその配下の自動タスクも止まります。役割の整理をするときは、先に自動タスクの一時停止または別役割への付け替えを行ってください。
:::

:::message alert
通知先を「LINE と Slack の両方」にしておくと、毎日同じ内容が両方に届きます。普段はどちらか片方で十分なケースが多いので、最初は **片方だけ** で運用して、必要に応じて増やすのが楽です。
:::

:::details スケジュール設定で迷ったときの目安
- 毎日チェックしたい情報（朝のブリーフィング、業界ニュース）→ 毎朝1回
- 週次レポート（競合分析、KPI 集計）→ 月曜朝 or 金曜夕方
- 投稿系（X 自動投稿）→ 平日朝・昼・夕方の3回 など、自分のリズムに合わせて
:::
