Teal は、AI が日常業務を理解して自律的に動く「AI × 自動化エージェント」です。Web ダッシュボードから操作するほか、LINE や Slack からも同じ AI を呼び出せるため、デスクでも外出先でも同じ作業が続けられます。さらに Google・Notion・X といった外部サービスとも連携でき、データの読み書き・通知・投稿までを Teal 上で完結できます。

このガイドは、Teal をはじめて触る方が **手を止めずに最後まで読み進められる** ことを目的にしています。製品の哲学を語るより、画面上の操作とその意味を順序立てて説明することを優先しました。

## このガイドでできるようになること

- Web ダッシュボード・LINE・Slack から Teal に話しかけられる
- カテゴリ・役割（ロール）・スキル・自動タスクの違いがわかる
- Google・LINE・Slack・Notion・X の5つの連携先を Teal に接続できる
- 自動タスクを設定して、AI が定期的に動くワークフローを組める

![Tealの全体像 — Webダッシュボードと5つの連携先の俯瞰図](/images/teal-user-guide/01-overview.png)

## 対象読者

- 個人事業主・フリーランスの方（業務効率化、SNS運用、スケジュール管理）
- 中小企業の担当者の方（営業・マーケティング、契約書作成、競合調査）
- 専門職の方（医療・福祉・製造業など、業界特化のテンプレートで業務を整えたい方）

専門知識は前提にしません。「LINE で AI と話したことがある」程度の経験があれば読み進められます。

## 章の歩み方

このガイドは2つの読み方を想定しています。

### A. 順番に読む（推奨）

第02章「5分で始める」から順に進むと、Teal の主要機能を一通り体験できます。各章はひとつのテーマに絞っているため、1章あたり10〜15分で読み終えられます。

### B. 必要な章だけ読む

すでに Teal を使っていて特定の機能だけ知りたい方は、目次から該当章へ直接飛んでください。各章は単体で完結するように書いてあります。

| 知りたいこと | 該当章 |
| --- | --- |
| 最短で動かしたい | 02. 5分で始める |
| Webダッシュボードの画面構成 | 03. Webダッシュボードの構成 |
| どの連携先から始めればいいか | 04. 5つの連携先 |
| AI に専門分野を持たせたい | 06. 役割（ロール） |
| 定期的に AI に動いてほしい | 08. 自動タスク |
| Googleカレンダーや Notion を接続したい | 09. 外部サービス連携 |
| うまく動かない | 11. つまずきやすいポイント |

## 表記について

本書では以下の表記で統一しています。

- サービス名は「**Teal**」（カナ表記の「ティール」は使いません）
- ソーシャルメディアは「**X（旧Twitter）**」
- Teal のような対話型 AI は「**チャットボット**」ではなく「**AIエージェント**」
- 外部サービスとの接続は「**連携先**」（Google・LINE・Slack・Notion・X の5種）。Teal への指示は Web ダッシュボード・LINE・Slack のいずれかから行います
- AI とのチャット単位は「**会話**」、AI の専門分野設定は「**役割**」、業種・機能別テンプレートは「**カテゴリ**」、AI が学習した能力は「**スキル**」、定期実行する自動化は「**自動タスク**」

:::message
本書は2026年5月時点の Teal の機能に基づいています。Teal は機能追加・改善が続いているため、画面表示が本書と異なる場合があります。
:::
