## この章でできるようになること

- カテゴリとサブカテゴリの違いがわかる
- 自分の業種に合うカテゴリを選べる
- カードをタップしてから AI が応答するまでの流れを理解できる

![10カテゴリのグリッド](/images/teal-user-guide/05-categories-grid.png)

## カテゴリは「最初の質問」をテンプレ化したもの

Teal の **カテゴリ** は、業種・目的別に整理されたテンプレート集です。Web ダッシュボードのメイン画面でカテゴリタブをタップすると、サブカテゴリ（タスクテンプレート）がカード形式で表示されます。

カードをタップすると、そのタスクに **最適化された最初の質問** を AI が投げかけてきます。「ターゲットは誰ですか？」「期間はいつまでですか？」のように、必要な情報を順番に聞いてくれるので、ユーザーは答えていくだけでタスクが進みます。

「最初に何を AI に聞けばいいかわからない」という状態を、カテゴリが先回りして解消してくれる、という設計です。

## 操作手順

### Step 1. カテゴリタブを選ぶ

Web ダッシュボードのメインエリア上部に、カテゴリタブが横並びで表示されています。

1. タブの中から、いまの目的に合うカテゴリをタップします
2. 該当カテゴリのサブカテゴリ（タスクテンプレート）がカード形式で並びます

カテゴリは10種類用意されています（後述の表を参照）。今後も追加・更新される予定です。

### Step 2. サブカテゴリのカードをタップする

サブカテゴリのカードをタップすると、新しい会話が即座に始まります。AI はそのタスク専用の質問を最初に投げてきます。

たとえば「マーケティング・企画」→「SNS集客」を選んだ場合、AI は次のような質問から始めます。

```text
SNS集客のサポートを始めます。
まずは前提を教えてください。

1. 業種・サービス内容
2. ターゲット層
3. いま運用しているチャネル
4. 達成したいゴール（フォロワー / 売上 / 認知）
```

順番に答えていくと、SNS 戦略・投稿文・ハッシュタグ提案へと会話が進んでいきます。

### Step 3. 会話の途中で別の話題に脱線してもOK

カテゴリで始まった会話でも、途中で「やっぱりターゲットを変えたい」「ハッシュタグだけ別案を10個」といった脱線は自由にできます。AI はカテゴリのテンプレートに **縛られず**、状況に合わせて応答を変えてくれます。

カテゴリはあくまで「最初の質問」を提示するだけで、その後の会話は通常のチャットとして進みます。

## カテゴリ一覧（10種類）

| カテゴリ名 | サブカテゴリの例 |
| --- | --- |
| 調べ物・情報整理 | 業界ニュース、法律・規制、補助金・助成金、商品比較、要約・翻訳 |
| 生活サポート | 日常のタスク管理、天気・服装アドバイス、筋トレメニュー作成 |
| 業務・事務 | 契約書ドラフト、社内通知文、データ入力・整理、在庫管理、備品発注 |
| 資金管理 | 資金調達戦略、予算管理、請求書作成 |
| マーケティング・企画 | SNS集客、コンテンツ作成、広告文作成、競合分析 |
| スケジュール・会議 | カレンダー管理、会議議事録作成、リマインダー設定 |
| 営業資料 | 営業リスト作成、プレゼン資料作成、提案書ドラフト |
| IT・Web向け | コードレビュー、デバッグ支援、ドキュメント生成、API設計 |
| 医療・福祉向け | シフト表作成、患者説明資料、研修資料、事業計画・報告書、業務マニュアル |
| 製造・メーカー向け | 品質管理、生産スケジュール、安全マニュアル作成 |

カテゴリは今後も追加・更新される予定です。最新の一覧は Web ダッシュボードのメインエリアで確認してください。

## つまずきやすいポイント

:::message
カテゴリは「最初の質問のテンプレート」なので、同じカテゴリでも始めるたびに違う方向に会話が進みます。「前回と同じカテゴリで始めたのに、AI の返事が違う」と感じても、それは仕様どおりです。
:::

:::details カテゴリと「役割」の違い
- **カテゴリ**：最初の質問テンプレート集。1回の会話の出発点として使う
- **役割**：AI に持続的な専門分野を持たせる設定。複数の会話で同じ振る舞いを再現できる

「毎回同じ業種で会話を始める」場合はカテゴリより役割（第06章）を作ったほうが効率的です。
:::
