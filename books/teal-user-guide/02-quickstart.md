## この章でできるようになること

- Teal のアカウントを作成して、最初の会話を始められる
- LINE 公式アカウントから Teal を呼び出せる
- 「カテゴリ」をひとつ選んで、AI と最初のタスクを進められる

![5分で始めるための5ステップ](/images/teal-user-guide/02-quickstart-flow.png)

## なぜ最初に LINE から始めるのか

Teal には Web ダッシュボードのほか、LINE と Slack から話しかけられる経路もありますが、最初の体験には **LINE** をおすすめします。理由はシンプルで、すでに LINE を使っている方が多く、新しいアプリのインストールも、ログイン情報の入力も不要だからです。

LINE で会話の感触をつかんだあと、第03章で Web ダッシュボードに移ります。Web は機能が多いぶん最初の一歩としては情報量が多いため、まずは LINE で「AI に話しかける」体験から入る順番にしています。

## 操作手順

### Step 1. アカウントを作成する

1. ブラウザで [teal-me.io](https://teal-me.io) を開きます
2. 「サインアップ」のボタンをタップします
3. メールアドレスと、Google アカウントなど任意の認証方法でアカウントを作成します

:::message
このガイドでは、アカウント作成後の最初の体験として「LINE での会話」を扱います。Web ダッシュボードのプロフィール設定は第03章で詳しく扱うので、ここではスキップしても構いません。
:::

### Step 2. LINE 公式アカウントを友だち追加する

1. Web ダッシュボードのサイドバーから「外部連携」を開きます
2. 連携サービス一覧の中から「LINE」を選びます
3. 表示された QR コードをスマートフォンの LINE で読み取り、Teal の公式アカウントを友だち追加します

QR コードを使わずに、LINE アプリ内で「Teal」と検索しても見つかります。友だち追加が完了すると、LINE 上の Teal とのトークルームが開きます。

### Step 3. アカウント連携を済ませる

1. トークルームに表示される最初の案内メッセージのリンクをタップします
2. ブラウザが開き、Teal のログイン画面に遷移します
3. Step 1 で作成したアカウントでログインすると、LINE と Teal アカウントが紐づきます

ここまでで、LINE と Web ダッシュボードが同じユーザーとして紐づきました。LINE 側で残した会話は Web 側からも参照できますし、その逆もできます。

### Step 4. 最初の会話を試す

LINE のトークルームに、自由なメッセージを送ってみます。たとえば次のように話しかけます。

```text
今日の予定をまとめて教えて
```

```text
来週の月曜日に「企画書ドラフト」というタスクを作って
```

```text
最新の業界ニュースを3件ピックアップして
```

AI が応答を返します。Googleカレンダーや Gmail との連携をまだ済ませていない場合は、AI が「カレンダーを連携しますか？」と聞き返してくることもあります。連携手順は第09章で扱うので、ここでは「あとで」と返しても構いません。

### Step 5. カテゴリを選んでタスクを始める

LINE には Web ダッシュボードのようなカテゴリ一覧の画面はありませんが、AI に「カテゴリを使いたい」と直接伝えることで、対応するテンプレートを呼び出せます。たとえば次のように送ります。

```text
マーケティング・企画のカテゴリで、SNS集客の会話を始めたい
```

AI が SNS集客に最適化された質問（業種・ターゲット層・現在の運用チャネルなど）を投げかけてきます。あとは順番に答えていけば、最初のタスクが進行します。

## つまずきやすいポイント

:::message
LINE 友だち追加のあとに「アカウント連携」のステップを飛ばすと、LINE 側からの指示が Web ダッシュボードの会話履歴に反映されません。最初に必ず Step 3 のリンクを開いてログインしてください。
:::

:::message alert
Teal の LINE 公式アカウントは1つだけです。検索結果に似た名前のアカウントが複数表示されたときは、公式サイト [teal-me.io](https://teal-me.io) からたどれる QR コードを使ってください。
:::

:::details うまく進まないときの確認順
1. Step 1 でメール認証のリンクを開いていますか（迷惑メールフォルダも確認）
2. Step 2 で LINE 側に「友だち追加」の通知が届いていますか
3. Step 3 のリンクから開いた画面で、Step 1 と同じアカウントでログインしましたか

このいずれかで止まっていることがほとんどです。
:::

## 次の章への橋渡し

LINE での最初の会話に慣れたら、次の章で Web ダッシュボードの構成を見ていきます。Web は会話履歴の整理・カテゴリ一覧・自動タスクの設定など、機能の中心が集まっている画面です。
